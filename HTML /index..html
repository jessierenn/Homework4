Summary: Lattitude vs. X

This project is to analyze changes of the weather as closer to the equator. Firstly, we need to pull the data from the OpenweatherMap API to assemble a dataset on over 500 cities.
Secondly, we used Matplotlib to plot various aspects of the weather vs. latitude. Factors we looked at included: temperature, cloudiness, wind speed, and humidity.
